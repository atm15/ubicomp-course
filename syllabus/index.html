Grading

Students will be evaluated based on their mini projects/assignments, class participation, and reading summaries. Students are expected to have the readings and summaries completed prior to class (see class schedule). Note that class discussion will largely be based on these readings. Class participation includes submitting reading summaries prior to class and active engagement in class discussions. Also, students should inform the instructor(s) on any work travel or other commitments that might arise during the quarter.

Component Percentage
Assignments (5) 60%
Weekly reading summaries 15%
Participation 25%
